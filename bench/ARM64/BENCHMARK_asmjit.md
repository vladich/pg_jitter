| Date | 2026-03-03 |

## Methodology

- Each cell shows: **absolute time in ms** and **(% of No JIT)** for that PG version
- `<100%` = JIT is faster than interpreter, `>100%` = JIT is slower
- No JIT baseline is the interpreter time for that specific PG version
- All queries run with `jit_above_cost = 0` to force JIT on every query
